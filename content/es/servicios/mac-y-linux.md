## En qué te ayudamos

Tanto si tienes un Mac que ha dejado de funcionar bien como si quieres instalar Linux en un ordenador viejo y sacarle unos años más de vida, te ayudamos.

### macOS

- Configuración y optimización del sistema
- Resolución de problemas (lentitud, errores, aplicaciones que fallan)
- Migraciones entre ordenadores Mac
- Recuperación de archivos y gestión de Time Machine
- Consejos de mantenimiento preventivo

### Linux

- Recomendación de la distribución adecuada para tu nivel y necesidades
- Instalación y configuración completa
- Resolución de problemas de hardware, red o periféricos
- Optimización para uso creativo, ofimático o de servidor

## Servicio especial: instalación Linux a precio fijo

¿Tienes un portátil u ordenador viejo que va lento con Windows o macOS? Podemos formatearlo e instalar una distribución Linux sencilla y funcional a un precio cerrado.

Incluye: borrado de datos, instalación del sistema, configuración básica (red, periféricos, actualizaciones) y prueba de funcionamiento. El cliente nos envía el ordenador o lo trae a Barcelona.

**Consulta disponibilidad y precio para este servicio específico.**

## Servicio especial: recuperación de datos

El disco duro ha muerto y hay datos importantes que necesitas recuperar. Trabajamos con discos en buen estado físico y casos donde el problema es de archivos, particiones o sistema de archivos.

- **Precio: 150 € por TB de disco** (fracción proporcional)
- El cliente envía o trae el disco duro
- **La recuperación no está garantizada**: cobramos si recuperamos datos, pero ciertos tipos de daño físico no se pueden resolver sin laboratorio especializado
- Si el caso supera nuestras capacidades, te derivamos a un servicio profesional de recuperación de datos
