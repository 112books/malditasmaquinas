## En qué te ayudamos

El software creativo de calidad existe. Y muchos son gratuitos, abiertos y tan potentes como las alternativas de pago. El problema es que nadie te ha enseñado a sacarles partido, o que cuando tienes un problema no sabes por dónde buscar.

Aquí es donde entramos nosotros.

Trabajas con:

- **Inkscape** — ilustración vectorial, logotipos, diseño gráfico
- **GIMP** — retoque fotográfico, composiciones, exportación para web e impresión
- **Darktable** — revelado y gestión de fotografía RAW en flujo profesional
- **Kdenlive** — edición de vídeo, montaje, exportación
- **Ardour** — producción musical, edición de audio, masterización básica
- **Affinity Publisher, Designer, Photo** — alternativa profesional a Adobe para quien prefiere licencia perpetua

Podemos ayudarte con:

- Configuración inicial y optimización del software para tu sistema
- Problemas de instalación, rendimiento o compatibilidad
- Formatos de archivo, exportación, conversión y compatibilidad con terceros
- Flujos de trabajo: cómo organizar el proyecto para que no sea un caos
- Crear un manual paso a paso para una tarea concreta que tienes que repetir a menudo

## Lo que no hacemos

No hacemos formación. Si necesitas aprender a usar Inkscape desde cero, hay recursos excelentes en línea para eso.

Lo que sí hacemos es resolver problemas concretos y escribir manuales de procedimiento para tareas específicas de tu flujo de trabajo habitual. "Cada vez que exporto a PDF para impresión me sale diferente" es el tipo de cosa que resolvemos.

<div class="note-box">
<strong>Adobe no lo recomendamos</strong>
Si estás pagando la suscripción de Adobe y no estás satisfecho, háblanos. En la mayoría de casos existe una alternativa de calidad equivalente (o mejor) sin suscripción mensual. No recomendamos Adobe a nuevos clientes.
</div>
